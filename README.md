# CallControl Sample Project 📞

این پروژه یک نمونه پیاده‌سازی برای مدیریت تماس‌ها و دریافت وب‌هوک‌های مربوط به تماس (Call Commands) است که با استفاده از **C#** و **.NET** نوشته شده است.


## پیش‌نیازها (Prerequisites)
برای اجرای این پروژه، موارد زیر را روی سیستم خود نصب داشته باشید:
*   [.NET SDK](https://dotnet.microsoft.com/download) (نسخه استفاده شده در پروژه را چک کنید، مثلا نسخه 6 یا 8)
*   [Visual Studio 2022](https://visualstudio.microsoft.com/) یا VS Code
*   بسته‌ی **Newtonsoft.Json** (برای مدیریت فایل‌های JSON)


## ویژگی‌های پروژه (Features)

قابلیت پخش فایل متنی (Play_Text)
قابلیت پخش فایل صوتی در تماس (Play_Audio).
قابلیت انتقال خودکار تماس (AutomaticTransfer).
قابلیت صف تماس (Queue).
قابلیت گرفتن اعداد وارد شده توسط کاربر(Gather)


## نحوه کارکرد (Workflow)
این پروژه به عنوان "سرویسِ پردازش‌گر" عمل می‌کند:
1. **دریافت رویداد:** پنل "دفترشما" اطلاعات تماس (مثل شماره تماس‌گیرنده، وضعیت تماس و...) را به آدرس Webhook شما ارسال می‌کند.
2. **پردازش در کد:** کلاس‌های پروژه شما این اطلاعات را دریافت کرده، تحلیل می‌کنند (مثلاً تشخیص می‌دهند تماس مربوط به کدام بخش است).
3. **ارسال دستور (Command):** پروژه شما یک پاسخ JSON استاندارد به API "دفترشما" ارسال می‌کند (مثل پخش صوت یا انتقال تماس).
4. **تکرار:** این چرخه تا پایان مکالمه ادامه می‌یابد.

## مراحل راه‌اندازی در پنل دفترشما
برای اینکه کد شما توسط پنل فراخوانی شود، مراحل زیر را در پنل "دفترشما" انجام دهید:
1. از منوی توسعه‌دهندگان به بخش **CallControl** بروید.
2. آدرس Webhook خود را ثبت کرده و **Token** دریافت کنید.
3. در تنظیمات "تلفن گویا" یا "لیست خطوط"، حالت مدیریت تماس را روی **کنترل تماس از طریق وب‌هوک** تنظیم کنید.
4. آدرس Webhook که در مرحله قبل ثبت کردید را به خط مورد نظر اختصاص دهید.
